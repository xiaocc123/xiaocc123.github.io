# 2019年度计划（初版）



## 一.目标

### 1. 学业与技术

1. 功课踏实学习。
2. 掌握 Python 编程，能够用 python 做出一些项目。
3. 入门人工智能，掌握基本的学习算法，掌握利用 Tensorflow 进行实践 。

### 2. 能力提升

1. 熟练使用思维导图
2. 学会使用 Markdown 来作为写作的排版、
3. 能够利用课余时间来练习写作，如写博客、公众号等。

### 3. 生活

1. 加入不同的圈子，开拓眼界。
2. 自我认识，自我控制，自我激励。
